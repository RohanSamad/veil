# Phase 1 JSON artefacts (Schemas + Examples)

These files make the bot outputs **copy/paste-able**, **auditable**, and easy to validate.

## What’s in here

- **Schemas** (`*.schema.json`): the contract the bot must return.
- **Examples** (`*.example.json`): realistic sample payloads that match the schemas.

## Files

- `road_test.schema.json` + `road_test.example.json`
- `scenario_analysis.schema.json` + `scenario_analysis.example.json`
- `debrief.schema.json` + `debrief.example.json`
- `audit_event.schema.json` + `audit_event.example.json` (internal logging)
